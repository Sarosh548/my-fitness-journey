# My Fitness Journey Website

Personal fitness tracker — 57kg → 70kg · Jun 2026 – Jan 2027

## What's included

- **Dashboard** — weight stats, daily checklist, progress bar, weight chart
- **Today** — full day timeline, water tracker, weight logging
- **Workout** — full home + gym workout plans (weekly rotation)
- **Nutrition** — all meals with calories, Phase 1 & Phase 2
- **Progress** — weight chart, monthly targets, gained so far
- **Schedule** — weekly overview + daily time breakdown

All data saves automatically in your browser (localStorage). No backend needed.

---

## Deploy to Vercel (Free — 5 minutes)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Upload your files to GitHub
1. Click the **+** button → **New repository**
2. Name it: `my-fitness-journey`
3. Click **Create repository**
4. Click **uploading an existing file**
5. Drag and drop ALL these files/folders:
   - `index.html`
   - `vercel.json`
   - `css/` folder
   - `js/` folder
6. Click **Commit changes**

### Step 3 — Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click **Add New Project**
3. Find `my-fitness-journey` and click **Import**
4. Click **Deploy** (no settings needed to change)
5. Wait ~30 seconds...

### Step 4 — Your website is live!
Vercel gives you a URL like:
`https://my-fitness-journey.vercel.app`

Share it with anyone! It's fully public.

## How to use the website

1. Open your URL every morning
2. Go to **Today** and check off tasks as you complete them
3. Log your weight each day using the weight input
4. Tap water glasses as you drink
5. Check **Dashboard** to see your overall progress
6. Use **Workout** to see what exercises to do each day

---

## To update your name or personalise
Open `index.html` and change "FitnessJourney" to your name.
Open `js/data.js` to edit meals or workouts.
